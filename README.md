# 封筒在庫管理システム
封筒管理アプリについて

概要）
一度発送したが、住所不在などで届かなかった封筒をシステム上に登録し**在庫管理を行う**。
また、得意先からの指示データに合わせて**再発送・廃棄**などの指示を受ける
